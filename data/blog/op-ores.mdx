# Installation

### ^Minecraft Bedrock \[PC\]

You can simply download the .mcaddon file and run it to install the addon, or else you can download the Zip file and extract it in this directory (works for all windows users, just copy and paste this location in File Explorer) :

```
C:\Users\%UserProfile%\AppData\Local\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\behavior_packs
```

### ^Minecraft Pocket Edition

For installation in minecraft pocket edition you can download and run the .mcaddon file or you can [read this article](https://mcpedl.com/how-to-install-minecraft-pe-mods-for-android/) for installation via .Zip file.
